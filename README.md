## GPU-based Feature adaptive rendering of Loop Subdivision Surfaces ##

### Dependencies ###
DirectX 11

### Test Environment ###
This program has been tested on Windows7, and it was developed with Visual Studio 2010.

### Run ###
Open the GPUSubLoop.sln, and compile.
